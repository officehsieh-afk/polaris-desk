## 已修正的隱藏 Bug（2026-06-18）

| Bug | 現象 | 修正 |
|-----|------|------|
| `normalizeCompany` 欄位錯誤 | peer-toolbar 公司名全是 undefined → 選不到公司 | `raw.id→raw.ticker`、`raw.name→raw.company_name??raw.ticker` |
| `normalizeAlert` 欄位錯誤 | alerts 全欄位 undefined，title/source 顯示空白 | 對齊後端 `{event_id,ticker,severity,evidence[]}` 結構 |
| `researchAlerts/peerAlerts` origin filter | `.filter(a=>a.origin==="research/peer")` 永遠 0 筆 | 移除 filter，待 R3 在 alert 層級補 `origin` 後再還原 |
| `normalizeNotifications` unread | 讀 `raw.unread`，後端實際回 `unread_count` | 改 `raw.unread_count` |
| CompanySlot `c.name.includes` crash | `company_name: null` 時 throw TypeError | 改 `(c.name ?? "").includes(search)` |
